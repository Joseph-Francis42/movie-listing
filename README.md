# 🎬 MovieLand

A professional, light-themed Movie Search Library and Watchlist Manager built with **React**, **Vite**, and the **OMDb API**. It features a clean tactile grid background, category filtering, pagination, and a detailed overlay modal for movie cast/plot statistics.

🔗 **[Live Demo](https://joseph-francis42.github.io/movie-listing/)**

---

## ✨ Features

- **Instant Search**: Search movies, TV series, and episodes in real-time.
- **Watchlist Manager**: Star your favorite titles to save them to your local Watchlist.
- **Detailed Overlays**: Click any movie card to fetch cast details, box office stats, and plot summaries.
- **Category Filtering**: Quickly filter search results by Movies, Series, or Episodes.
- **Custom API Settings**: Change or update your OMDb API Key directly in the UI.

---

## 🚀 Setup & Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Joseph-Francis42/movie-listing.git
   cd movie-listing
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start local development server:**
   ```bash
   npm run dev
   ```

4. **Build for production:**
   ```bash
   npm run build
   ```
